# CuraOS Documentation

Welcome to the CuraOS customer and operator documentation. CuraOS is a
composable, self-hosted-first care platform: a generic neutral core with opt-in
vertical overlays (Health, Education, ERP).

This site is the **external** documentation surface: install guides, integration
references, API reference, and operator runbooks. It is built as a fully offline,
air-gap-renderable static site with client-side search.

> This `examples/content/` tree is an in-repo fixture so the build is
> exercisable standalone. Real authored content is staged from the workspace
> docs-content mirror at build time.

## Sections

- [Install](install/index.md): deploy CuraOS across cloud, on-prem, hybrid, and air-gap profiles.
- [Integration](integration/index.md): connect external systems and overlays.
- [Operator runbooks](operations/index.md): day-2 operations, alerts, on-call.
- [API reference](api/index.md): generated TypeScript API documentation.
